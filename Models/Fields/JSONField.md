# JSONField

## JSONField

* Installation

  ```shell
  pip install jsonfield
  ```

* Usage

  ```python
  from jsonfield import JSONField

  user_info = JSONField(_(u'user_info'), blank=True, null=True, help_text=u'User Info, JSON Format')
  ```
